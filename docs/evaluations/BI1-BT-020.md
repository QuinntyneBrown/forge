# BT-020 — Set leaderboard opt-in

- Default `LeaderboardOptIn` is `false`. PUT `true` → 204 → GET /api/me reflects `true`. PUT `false` → 204 → GET /api/me reflects `false`.

- Handler: depends on `IAppDbContext` + `ICurrentUser`; loads the user, sets `LeaderboardOptIn`, saves
- `[Authorize] [HttpPut("leaderboard-opt-in")]` on `ProfileController` returning `204 NoContent`
